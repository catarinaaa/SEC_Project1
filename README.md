# sec2019
Highly Dependable Systems project from IST.

## Requirements
You must have installed the following tools:\

* Maven

## How to Compile and Run
**Library**
```
cd Library
mvn clean install
```

**Server**
```
cd Server
mvn clean compile exec:java
```
**Client**\
On another shell:
```
cd Client
mvn clean exec:java
```
